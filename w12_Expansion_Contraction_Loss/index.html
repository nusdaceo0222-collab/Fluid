확대관·축소관 부차손실 시뮬레이터

관 지름이 갑자기 변할 때 발생하는 부차손실수두를 계산합니다. 확대관에서는 급확대 손실을 hm = (V₁−V₂)²/2g로 계산하고, 축소관에서는 hm = K V₂²/2g 형태로 계산합니다.

확대손실
확대관: hm = (V₁−V₂)²/2g
hm = (3.98−0.99)²/(2×9.81) = 0.46 m
ΔP = ρghm = 4.5 kPa
확대관에서는 빠른 유동이 갑자기 넓은 관으로 들어가면서 박리와 와류가 발생하고, 속도차가 손실로 전환됩니다.

시각화

배관 내 유동 손실 영역 에너지 손실 속도·치수 표시
확대관·축소관 손실은 배관이 길어서 생기는 주손실이 아니라, 단면 변화 때문에 생기는 부차손실입니다. 유속이 커질수록 속도수두가 커져 손실도 증가합니다.
