직관·곡관·밸브 전체 배관손실 종합 시뮬레이터

직관의 주손실과 곡관·밸브의 부차손실을 합산하여 전체 손실수두와 압력손실을 계산합니다. 앞에서 학습한 달시-바이스바흐식, 무디선도, 부차손실식을 종합 적용하는 시뮬레이터입니다.

이 종합 시뮬레이터는 앞 단계의 계산을 모두 연결합니다. 무디선도에서 f 결정 → 직관 주손실 계산 → 곡관·밸브 K 합산 → 전체 손실수두 계산 순서로 해석합니다.
